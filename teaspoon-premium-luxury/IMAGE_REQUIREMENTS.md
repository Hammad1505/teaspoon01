## üìù Image Optimization Tips

### Before Uploading:
1. **Resize images** to recommended dimensions
2. **Compress images** using tools like:
   - TinyPNG (https://tinypng.com)
   - Squoosh (https://squoosh.app)
   - JPEG Optimizer
3. **Convert to JPEG** if using PNG (for photos)
4. **Use WebP** format for better compression (optional)

### Photography Tips:
- Use natural lighting when possible
- Keep focus on the food/subject
- Show vibrant colors (especially for food)
- Include garnishes for food photos
- Use clean, uncluttered backgrounds
- Shoot from slightly above (45¬∞ angle) for food
- Include props (cutlery, napkins) for context
